Descrierea claselor

Mapper

Citeste linii din stdin, curata cuvintele (lowercase, fara punctuatie) si emite perechi (cuvant, doc_id) pe stdout. Obtine document_id din variabila de mediu.

mapper_inverted_index.py

Reducer

Primeste perechi sortate din stdin, le grupeaza pe cheie (cuvant), deduplica doc_id-urile si emite (cuvant, [doc1, doc2, ...]) pe stdout.

reducer_inverted_index.py

LocalRunner

Simuleaza intregul pipeline MapReduce in Python pur, fara Hadoop. Apeleaza intern logica de mapare, sortare si reducere, scrind rezultatul in fisier.

run_local.py

InvertedIndex

Structura de date finala: dictionar word → [doc_id_1, doc_id_2, ...]. Reprezinta output-ul algoritmului, scris in fisier text.

output_local.txt / part-00000

HadoopStreaming

Componenta externa Apache Hadoop care orchestreaza rularea distribuita a mapper-ului si reducer-ului prin hadoop-streaming*.jar pe un cluster HDFS.

hadoop-streaming*.jar

Relatii intre componente

SursaRelatieDestinatieDescriere

Reducer → produce InvertedIndex Agregate perechile in indexul final
LocalRunner → apeleaza Mapper Invoca logica de mapare per document
LocalRunner → apeleaza Reducer Invoca logica de reducere pe perechile sortate
LocalRunner → scrie InvertedIndex Persistente rezultat in output_local.txt
HadoopStreaming → ruleaza Mapper Lanseaza ca subprocess per input split
HadoopStreaming → ruleaza Reducer Lanseaza dupa faza de sort/shuffle
HadoopStreaming → scrie InvertedIndex Scrie part-00000 in HDFS
